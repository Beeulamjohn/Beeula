# 🌱 Plant Solutions Website  

A simple **HTML + CSS** landing page project that showcases services, projects, and team updates.  
This project is beginner-friendly and demonstrates the use of **Flexbox, absolute positioning, and styled buttons**.  

---

## 📸 Preview  

plant website.png

---

## 📂 Project Structure  

```

project-folder/
│── index.html        # Main HTML file
│── ABC.css           # CSS stylesheet
│── plant2.png        # Banner/hero image
│── rocket.jpg        # Logo 1
│── magic.png         # Logo 2
│── gift.png          # Logo 3
│── phone.png         # Logo 4

```

---

## 🚀 Features  

✅ **Header Navigation Bar** with Home, Services, Projects, Blog, Team, and Pages.  
✅ **Hero Section** with a background image, tagline, and a call-to-action button.  
✅ **Introduction Section** describing the importance of plants.  
✅ **Feature Container** using **Flexbox** with four service highlights:  

1. ✈️ Easy Flight Search  
2. 🏨 Get Hotel Offers  
3. 🎁 Holiday Packages  
4. 📞 Dedicated Support  

---

## 🎨 Styling (ABC.css)  

- **Flexbox Layout** for header & feature section.  
- **Absolute Positioning** for centered hero text.  
- **Custom Button Styling** with rounded corners and `blueviolet` background.  
- **Consistent Indentation & Alignment** for icons and text.  

---

## 🛠️ How to Run  

1. Clone or download this repository.  
2. Ensure all image files (`plant2.png`, `rocket.jpg`, `magic.png`, `gift.png`, `phone.png`) are in the same directory as `index.html`.  
3. Open `index.html` in any modern browser (Chrome, Edge, Firefox, etc.).  

---

## 📌 Notes  

- If images don’t load, check that filenames match exactly.  
- You can customize fonts, colors, and layout in `ABC.css`.  
- Add your own content (text/images) to personalize the website.
